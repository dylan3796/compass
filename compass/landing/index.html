How it works

Connect → Define → Reconcile → Act.

Two MCP connections and an outcome definition.

STEP 1

Connect your agent

One MCP config line. Claude Code, Codex, or custom SDK agents. Compass captures runs, cost, and completion.

STEP 2

Define the outcome

Bind each agent to the system where its outcome lives — Salesforce, Zendesk, Stripe. Set the unit value and the projection. Read-only.

STEP 3

Reconcile weekly

Delivered value, read from the system of record, against plan. Measured — not self-reported by the agent.

STEP 4

Act

Fix the prompt, clone the winner, add a guardrail. Watch the value line.
