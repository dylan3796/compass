04The Fix List — ranked by monthly impact

#1 · RIGHT_SIZE_MODEL · SUMMARIZER

Summarizer runs an Opus-tier model on a summarization task class where a fast-tier model rates within ~3% on the audit sample. Est. $76/mo at current volume. One-line config change; verify on 50 runs.

#2 · ADD_GUARDRAIL · SUPPORT

Completion-checker guardrail (max 3 revisions, 2k-token stall detection) to break revision loops. Direct waste recovered: $3.40/mo — but the projection gap it closes is worth ~$1,900/mo in deflection value at projected volume.

#3 · TRIM_CONTEXT · SUMMARIZER

Structured input (abstract + key sections) instead of whole documents. Est. $89/mo on the current model — overlaps with #1; if both ship, combined savings are capped at Summarizer's $96/mo spend. Counted once in the summary.

#4 · ONBOARD SHADOW AGENT · DIGEST-BOT

Move off the personal API key, onto the platform account with logging. Risk fix, not a cost fix: untracked spend (~$28/mo est.) and no audit trail.

#5 · PROMPT_REGRESSION · OUTBOUND

Quality fell ~40% after the "punchy rewrite" prompt change three weeks ago. Roll back to the structured v1 prompt; A/B the rewrite properly if the reply-rate hypothesis still matters.
